## 질문 목록
Q. child_status는 누가 가지고 있나?
A. 부모와 자식이 둘 다 같은 child_status 객체를 가르킴

Q. 부모 스레드인지는 어떻게 아나?
A. 자식 생성은 부모가 실행 중일 때 일어남, 따라서 현재 시점에서 실행되고 있는게 부모 스레드이고,
새롭게 생성될 스레드가 자식 스레드

Q. 현재 상태는 무엇이고, 
A. Pintos 부팅 -> 커널 초기화 -> 파일 시스템 초기화 -> 스케줄러 시작 -> 커널 main thread가 실행 중
 process_create_initd()호출 -> thread_create()로 담당 커널 스레드 1개 생성
